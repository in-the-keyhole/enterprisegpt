Welcome to the EnterpriseGPT Node.js (Express) Back-end

This server acts as the back-end for our EnterpriseGPT React Front-end application. It is primarily used as the API middleman between the React front-end and the OpenAI ChatGPT API calls we will be making.

API Endpoints

Our server makes use of the following OpenAI GPT API endpoints:

https://api.openai.com/v1/chat/completions

This endpoint is used to generate chat completions.
